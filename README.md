# Daily Briefing

A personal learning log: two new things a day across AI &amp; gaming, the games &amp; esports industry, leadership &amp; negotiation, and live events &amp; immersive.

## How it works

- `index.html` is a single, self-contained page (HTML + CSS + JS). It fetches `fabian-learning.md` and renders each dated edition as a styled card.
- `fabian-learning.md` is the content. A new edition is added to the top each morning.
- Netlify watches this repo and redeploys automatically on every commit.

## Updating

Edit `fabian-learning.md` (newest edition on top), commit, and the live site refreshes within a minute or two.

No build step. Publish directory is the repo root.
